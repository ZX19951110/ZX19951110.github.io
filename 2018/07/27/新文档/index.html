ThoughtWorks2019校园招聘作业

逻辑实现

根据题干，二维迷宫本质上是二维数组，根据迷宫的性质，给定参数rows，cols，则迷宫为一个(2*rows+1) * (2*cols+1)大小的二维数组，之后的连通等功能均在此二维数组上操作。

根据题干信息需要四个异常类，其中int类型转换异常不需定义。

根据用户输入的命令，需要使用正则表达式对用户输入进行校验，若不符合则抛出异常。

根据第一行命令初始化迷宫后需要根据第二行命令对迷宫进行连通。首先需要对参数进行校验，两个参数分别对应坐标的y与x(以左上角为零点向下右延展)，如果两个坐标无法连通，则抛出异常，若坐标超出范围，同样抛出异常。由题可知需连通的墙坐标应该为 y = (y1+y2)/2 x = (x1+x2),所以需要连通的位置应为grid[y-1][x-1]。

连通完成后返回迷宫对应的字符串并打印出来。

Java代码实现

主函数

主函数主要是对输入参数的格式以及坐标连通性等条件进行校验，若校验失败则抛出异常并输出相应结果。之后解析输入命令并作为构造参数传入CreateGrid类并实例化，最后打印出得到的连通迷宫。

import Exceptions.FormatException;
import Exceptions.MazeFormatException;
import Exceptions.OutOfRangeException;
import Grid.CreateGrid;

import java.util.*;
import java.util.regex.Pattern;

public class Main {
public static void main(String args[]){
    Scanner scan = new Scanner(System.in);
    String[] buildCMD = new String[2];
    for (int i = 0; i < 2; i++){
       buildCMD[i] = scan.nextLine();
    }
    scan.close();
    try {
        //check command format
        if (!Pattern.matches("(-)?[A-Za-z0-9]+\\s(-)?[A-Za-z0-9]+", buildCMD[0]) || !Pattern.matches("((-)?[A-Za-z0-9]+\\,(-)?[A-Za-z0-9]+\\s(-)?[A-Za-z0-9]+\\,(-)?[A-Za-z0-9]+\\;)+", buildCMD[1]+';')){
            throw new FormatException();
        }
        //check roadGrid
        int rows = Integer.parseInt(buildCMD[0].split(" ")[0]);
        int cols = Integer.parseInt(buildCMD[0].split(" ")[1]);
        if (rows <= 0 || cols <= 0){
            throw new OutOfRangeException();
        }
        //parse linkCMD
        String[] linkComands = buildCMD[1].split(";");
        ArrayList<HashMap> linkList = new ArrayList<>();
        for (String s : linkComands) {
            HashMap<String, Integer> linkCoordsMap= new HashMap<String, Integer>();
            String[] coords = s.split(" ");
            int protoCoord1Y = Integer.parseInt(coords[0].split(",")[0]);
            int protoCoord1X = Integer.parseInt(coords[0].split(",")[1]);
            int protoCoord2Y = Integer.parseInt(coords[1].split(",")[0]);
            int protoCoord2X = Integer.parseInt(coords[1].split(",")[1]);
            if ((protoCoord1X != protoCoord2X && protoCoord1Y != protoCoord2Y)){
                throw new OutOfRangeException();
            }
            if ((Math.abs(protoCoord1X - protoCoord2X) > 1) || (Math.abs(protoCoord1Y - protoCoord2Y) > 1) ){
                throw new MazeFormatException();
            }
            int coord1X = 2 * protoCoord1X + 2;
            int coord1Y = 2 * protoCoord1Y + 2;
            int coord2X = 2 * protoCoord2X + 2;
            int coord2Y = 2 * protoCoord2Y + 2;
            linkCoordsMap.put("coord1X", coord1X);
            linkCoordsMap.put("coord1Y", coord1Y);
            linkCoordsMap.put("coord2X", coord2X);
            linkCoordsMap.put("coord2Y", coord2Y);
            linkList.add(linkCoordsMap);
        }
        //main
        CreateGrid grid = new CreateGrid(rows, cols, linkList);
        System.out.println(grid.render());
    }
    catch (NumberFormatException e){
        System.out.println("Invalid number format.");
    }
    catch (OutOfRangeException e){
        System.out.println("Number out of range.");
    }
    catch (MazeFormatException e){
        System.out.println("Maze format error");
    }
    catch (FormatException e){
        System.out.println("Incorrect command format.");
    }

}

}

构建迷宫类

首先需要初始化参数，生成未连通的迷宫，其中当数组下标 x % 2 ==0 || y % 2 == 0时，作为Cell，然后对迷宫进行连通。

public class CreateGrid {
private int cols, rows, renderGridX, renderGridY;
private int[][] renderGridArr;
public CreateGrid(int rows, int cols, ArrayList<HashMap> linkList){
    //set parameters
    this.cols = cols;
    this.rows = rows;
    this.renderGridX = 2 * this.cols + 1;
    this.renderGridY = 2 * this.rows + 1;
    this.renderGridArr = new int[this.renderGridY][this.renderGridX];
    // Build Grid with out link
    for (int i = 0; i < this.renderGridY; i += 1){
            for (int j = 0; j < this.renderGridX; j += 1){
                if (j % 2 == 0 || i % 2 == 0){
                    this.renderGridArr[i][j] = 0;
                }
                else {
                    this.renderGridArr[i][j] = 1;
                }
        }
    }
    this.linkRoad(linkList);
}

连接迷宫

主函数解析第二行参数得到连接坐标数组，并逐个根据字符串分割解析成int型坐标，并将坐标由0置为1。

// link road
public void linkRoad(ArrayList<HashMap> linkString) {
    for (HashMap<String, Integer> mp : linkString) {
        int coord1X = mp.get("coord1X");
        int coord1Y = mp.get("coord1Y");
        int coord2X = mp.get("coord2X");
        int coord2Y = mp.get("coord2Y");
        if (coord1Y == coord2Y) {
            int roadX = (coord1X + coord2X) / 2;
            this.renderGridArr[coord1Y - 1][roadX - 1] = 1;
        }
        if (coord1X == coord2X) {
            int roadY = (coord1Y + coord2Y) / 2;
            this.renderGridArr[roadY - 1][coord1X - 1] = 1;
        }
    }

打印迷宫

根据二维数组，1则为[R]，0则为[W]

public String render(){
   String row = "";
       for (int[] i: this.renderGridArr){
           for (int j: i){
               if (j == 0){
                   row += "[W]";
               }
               else {
                   row += "[R]";
               }
           }
           row += ("\n");
       }
       return row;
   }
